文字处理

列表：

  • 无序列表项：使用+、-、*实现,加号的使用就是现在所见的，减号为方块；前后错落使用空格区分。当前使用了css样式美化，显示有所出入
  • 有序项列表：使用1. 2.实现，就是如1.2生成文章所示。

字体：

  1. 斜体 *斜体*；
  2. 加粗 **加粗** 或 <b>加粗</b>
  3. 斜体加粗 ***斜体加粗***
  4. 改字体大小
    • font的3大小:<font size="3">font</font>
    • span的15像素大小<span style="font-size:15px">span</span>

文本标记：

  1. 删除线 删除线
  2. 下划线 <u>下划线</u>；
  3. 波浪线 <span style="text-decoration-line: underline; text-decoration-style: wavy;">波浪线</span>
  4. 文字颜色
    1. 字颜色：
       + <font color=red>font:</font> `<font color=red>font</font>`
      • span: <span style="color:yellow; font-size:15; font-weight:bold">span:</span>
    2. 文字背景:
      • span: <span style="color:white; background:black">span:</span>
